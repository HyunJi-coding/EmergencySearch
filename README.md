<h2>응급실 병원 통합 검색<br></h2>
사용자가 검색한 위치를 기반으로 가까운 응급실을 추천하고 응급실의 상세 정보를 제공합니다.<br>
응급실의 상세 정보와 함께 즐겨찾기 및 후기 기능을 제공합니다.
<br>
<h3>1. 기술 스택<br></h3>
Java: 11<br>
Spring Boot: 2.6.7<br>
database: MariaDB, Redis<br>
build: Gradle<br>

• 소셜 로그인<br>

• 즐겨찾기 관리<br>
• 후기 작성 및 관리<br>
